ws-to.me			WS-TO-ME v1.0			ctran 

DESCRIPTION
	A public websocket relay server. Stream data 
	from client-to-client without either node knowing 
	of each other's existence with plain JS websockets.
	
	Thank you pipe-to-me for the inspiration.

SESSION URL
	ws://{{index . "host"}}/{{index . "sessionKey"}}

PUBLISH EXAMPLE
	// plain js
	socket = new WebSocket("ws://{{index . "host"}}/{{index . "sessionKey"}}");
	socket.send("Hello room {{ index . "sessionKey" }}");

SUBSCRIBE EXAMPLE
	// plain js
	socket = new WebSocket("ws://{{index . "host"}}/{{index . "sessionKey"}}");
	socket.addEventListener("message", (e) => {
		console.log(`FROM RELAY: ${e.data}`);
	});
